# Thunder JavaScript Runtime

A small JavaScript interpreter built from scratch in Python for Thunder Hackathon 2.

## Goal

Execute an educational subset of JavaScript using this pipeline:

JavaScript source -> Lexer -> Parser -> AST -> Interpreter -> stdout

## Rules

This project does not use Node.js, QuickJS, V8, js2py, Python `eval()`,
Python `exec()`, or any existing JavaScript engine.

## Status

Project setup is complete. Interpreter development has not started yet.
